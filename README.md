# Python Basic Calculator

This is a simple Python calculator that performs the four basic operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## How it works
The program:
1. Asks the user to enter two numbers
2. Asks the user to choose an operation
3. Displays the result or an error message if the operation is invalid

## Features
- Handles division by zero
- Supports basic user input
- Written for beginners learning Python
